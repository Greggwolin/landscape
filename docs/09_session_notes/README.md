# Daily Syncs (CANONICAL)

This folder holds the **nightly automated daily-sync files** generated by the daily-brief skill. Each dated file summarizes commits, schema changes, feature work, bug fixes, and notable decisions for that day.

## How files land here

Generated automatically each night. **Do not hand-edit** — the next nightly run overwrites or extends per its own logic. If you need to add a user-authored note tied to a session, append it to `docs/daily-context/session-log.md` instead (newest first).

## Canonical routing for status / history

| Source | Purpose |
|---|---|
| `docs/09_session_notes/` (this folder) | Nightly automated daily syncs — CANONICAL recent state |
| `/landscape/CLAUDE.md` § "Alpha Readiness Assessment" | High-level status, updated session-by-session |
| `docs/daily-context/session-log.md` | User-invoked session-log entries (the `Document` command) |
| `docs/session-notes/` (no prefix) | HISTORICAL — old location, March 9 → April 17, 2026 only |
| `docs/09-session-notes/` (hyphen) | HISTORICAL — older session material (prompts, prototypes, AI chats), 2025 only |
| `docs/00-overview/IMPLEMENTATION_STATUS_3-8-26.md` | HISTORICAL — superseded snapshot frozen at April 17, 2026 |

If you're looking for "what shipped today / this week," read the most recent dated files in this folder.
