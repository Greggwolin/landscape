## Example Query

```sql
-- Get complete rent roll
SELECT
  s.space_number,
  t.tenant_name,
  s.rentable_sf,
  s.space_type,
  s.space_status,
  l.lease_type,
  br.base_rent_psf_annual as current_rent_psf,
  br.base_rent_annual as annual_rent,
  l.lease_expiration_date
FROM landscape.tbl_cre_space s
LEFT JOIN landscape.tbl_cre_lease l ON s.space_id = l.space_id
LEFT JOIN landscape.tbl_cre_tenant t ON l.tenant_id = t.tenant_id
LEFT JOIN landscape.tbl_cre_base_rent br ON l.lease_id = br.lease_id
  AND br.period_start_date <= CURRENT_DATE
  AND br.period_end_date >= CURRENT_DATE
WHERE s.cre_property_id = 3
ORDER BY s.rentable_sf DESC;
```
