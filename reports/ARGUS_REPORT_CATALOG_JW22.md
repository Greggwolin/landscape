## LANDSCAPE IMPLEMENTATION ARCHITECTURE

### Core Design Principles

1. **Universal Data Model** — All reports query the same universal container/budget/period structure
2. **Property-Type Polymorphism** — Reports adapt based on `project_type_code`
3. **Template-Based Generation** — SQL Query + Layout Definition + Formatting Rules + Export Options
4. **Parameter-Driven Flexibility** — Date Range, Container Filter, Grouping, Detail Level, Comparison

### Database Schema

```sql
-- Report definitions table
CREATE TABLE tbl_report_definition (
  report_id BIGSERIAL PRIMARY KEY,
  report_code VARCHAR(50) UNIQUE NOT NULL,
  report_name VARCHAR(200) NOT NULL,
  report_category VARCHAR(50),
  property_types VARCHAR[],
  report_tier VARCHAR(20),
  is_active BOOLEAN DEFAULT TRUE,
  sort_order INTEGER
);

-- Report templates (SQL + layout)
CREATE TABLE tbl_report_template (
  template_id BIGSERIAL PRIMARY KEY,
  report_id BIGINT REFERENCES tbl_report_definition(report_id),
  template_name VARCHAR(200),
  sql_query TEXT,
  layout_definition JSONB,
  formatting_rules JSONB,
  export_options JSONB,
  is_default BOOLEAN DEFAULT FALSE
);

-- Report parameters
CREATE TABLE tbl_report_parameter (
  parameter_id BIGSERIAL PRIMARY KEY,
  report_id BIGINT REFERENCES tbl_report_definition(report_id),
  parameter_name VARCHAR(100),
  parameter_type VARCHAR(50),
  default_value TEXT,
  is_required BOOLEAN DEFAULT FALSE,
  display_order INTEGER
);

-- User report history
CREATE TABLE tbl_report_history (
  history_id BIGSERIAL PRIMARY KEY,
  report_id BIGINT REFERENCES tbl_report_definition(report_id),
  project_id BIGINT REFERENCES tbl_project(project_id),
  user_id VARCHAR(100),
  parameters JSONB,
  generated_at TIMESTAMP DEFAULT NOW(),
  export_format VARCHAR(20),
  file_path TEXT
);
```
